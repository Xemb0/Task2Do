# Task2Do

Task2Do is a task manager application built with MongoDB Atlas, Express.js, and React. It allows users to manage tasks with features such as adding, editing, and deleting tasks.

## Features

- Add new tasks with title, description, and due date.
- Edit existing tasks.
- Delete tasks.
- Responsive design for usability on both desktop and mobile devices.

## Technologies Used

### Frontend

- React.js
- React Router DOM
- Material-UI
- Axios for API requests
- Emotion for styling
- Vite for development and bundling

### Backend

- Node.js with Express.js
- MongoDB Atlas for database storage
- Mongoose for MongoDB object modeling
- CORS for enabling cross-origin requests
- Dotenv for environment variables
